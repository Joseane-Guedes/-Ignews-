# 🔧 Run Locally

Clone the project

```bash
  git clone git@github.com:Joseane-Guedes/ignews.git
```

Go to the project directory

```bash
  cd ignews
```

Install dependencies

```bash
  yarn 
```

Start the server

```bash
  yarn dev 
```
